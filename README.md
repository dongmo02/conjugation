Dieses Projekt ist ein Konsolenprogramm in Python zur Konjugation deutscher Verben in verschiedenen Zeitformen.

Das Programm unterstützt:

Präsens

Futur I

Präteritum

Es enthält eine vereinfachte Behandlung von:

Trennbaren Verben

Modalverben

Hilfsverben

Regulären (schwachen) Verben

Das Projekt wurde im Rahmen des Selbststudiums zur Vertiefung von Python, String-Verarbeitung und Programmstrukturierung entwickelt.
